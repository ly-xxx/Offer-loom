## Release note

- when publishing to GitHub, keep the upstream links in the README or release notes
- if you add more public guide or interview repositories later, add their attribution here as well
- before importing any third-party repository, review its license, attribution requirements, and redistribution boundary

## Runtime extension

The repository contains local source directories that can be indexed immediately. At runtime, users can still add more content through:

- local directory import
- Git repository clone
- the in-app `+` button for manual interview imports
