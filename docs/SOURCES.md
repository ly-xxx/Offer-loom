## 发布建议

- 发布到 GitHub 时，请在 README 或发布说明中保留这些公开来源的上游链接
- 如果你继续加入新的公开文档或面经仓库，也建议在本文件中补充来源和链接
- 引入第三方仓库前，请自行检查其许可证、署名要求和再发布边界

## 说明

仓库内存放的是可直接索引的本地来源目录；运行时也支持继续追加：

- 本地目录导入
- Git 仓库克隆
- 站内 `+` 按钮手动导入新面经
