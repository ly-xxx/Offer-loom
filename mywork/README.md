# MyWork Placeholder

把你自己的论文、代码、笔记、notebook、实验记录放到这个目录里。

推荐结构：

```text
mywork/
├── README.md
├── project-a/
│   ├── README.md
│   ├── paper.pdf
│   ├── notes/
│   ├── notebooks/
│   └── src/
└── project-b/
    ├── README.md
    ├── docs/
    └── code/
```

说明：

- 这个目录默认不会被提交到 GitHub，当前文件是唯一保留的占位说明。
- 首次使用时，如果你还没有准备 `mywork`，也可以先只用仓库内公开文档和题库建索引。
- 如果你的真实工作集不在这里，也可以在站内设置页切换到任意本地目录。
